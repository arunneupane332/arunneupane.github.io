Ticketing System

  • Frontend: Thymeleaf
  • Backend: Java Spring Boot (RestAPI)
  • Database: MySQL

Project detail

Designed and developed a comprehensive Ticketing System for Cell Pay Private Limited, utilizing (RestAPI and Web part) Spring Boot for backend services, Thymeleaf for frontend Web templating, and MySQL for database management. This sophisticated platform facilitates efficient handling of customer support tickets, tracking issues from creation to resolution, and maintaining detailed records of customer interactions. The system demonstrates advanced full-stack development capabilities, highlighting proficiency in integrating backend and frontend technologies to deliver a seamless and user-friendly support solution.
